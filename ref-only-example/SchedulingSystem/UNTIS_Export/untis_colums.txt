Exp/Imp Time Requests (016) Sequence of fields in the DIF-file GPU016.TXT

 

GPU016.TXT DIF-file Time requests (Period Time Requests) for teachers, classes, rooms and subjects:

Number        Field

1        Element Short Name

2        Type of element (T=Teacher, C=Class, R=Room, S=Subject)

3        Day

4        Period

5        Time request (-3 to + 3)

A time request with the value 0 deletes a potential time request in this period.

 

You can delete all time requests of an element by adding Day = 0 and Period = 0 to this element.
